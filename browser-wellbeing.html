Browser Wellbeing: A Chrome Extension for Mindful Browsing

Overview:

In an age of constant digital distraction, this project is a custom-built Chrome extension designed to help users regain control of their browsing habits. The Browser Wellbeing Extension promotes digital health by allowing users to set time limits on distracting websites, receive gentle reminders to take breaks, and block sites once the allocated time is up. It's a practical tool for anyone looking to improve focus and productivity.

Project Description:

This extension was developed from scratch to provide a simple yet powerful solution for time management directly within the user's browser. It features an intuitive popup menu for quick actions and a comprehensive options page for personalization, allowing users to define their own list of distracting websites and corresponding time limits.

The core of the project was built with vanilla JavaScript and key browser technologies:

  • JavaScript (ES6): Powers all the core logic, including the timer functionality, website tracking, and dynamic manipulation of the extension's UI components.
  • Chrome Extension APIs: Leveraged key APIs like chrome.storage to persistently save user settings (blocked sites, time limits) across browsing sessions and chrome.tabs to monitor user activity and enforce the blocking rules.
  • HTML5 & CSS3: Used to structure and style the user-facing components, including the clean and straightforward popup menu and the detailed options page.
