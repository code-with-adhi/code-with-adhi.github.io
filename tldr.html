TLDR News: An AI-Powered Full-Stack News Summarizer

Overview:

TLDR News is a full-stack web application designed to combat information overload. It fetches top headlines from around the world, allows the user to select an article, and then uses a custom-built web scraper and the Google Gemini AI to provide a concise, easy-to-read summary. This project demonstrates a complete development cycle, from frontend interface design to a complex, containerized backend service.

Project Description:

This application solves a common problem: not having enough time to read full news articles. It features a React-based frontend that communicates with a powerful Node.js backend. The backend is responsible for fetching news, performing robust web scraping, and interfacing with the AI model for summarization.

The project was built with a focus on scalability and modern development practices, utilizing the following technologies:

  • React.js & React Router: The client-side is a dynamic Single Page Application (SPA) that provides a seamless user experience for browsing headlines and viewing summarized articles without page reloads.
  • Node.js & Express.js: The backend server handles all API requests, managing the logic for fetching news, scraping external sites, and processing summarization requests.
  • Puppeteer & Readability.js: A sophisticated web scraping service was built to navigate to external news articles, bypass anti-scraping measures, and intelligently extract the core article content, stripping away ads and boilerplate.
  • Google Gemini API: Integrated to provide high-quality, context-aware summaries of the scraped article text, turning long-form content into a quick digest.
  • Docker & Render: The entire backend is containerized using Docker, ensuring a consistent and isolated environment. It is deployed as a web service on Render, showcasing modern DevOps and cloud deployment skills.
